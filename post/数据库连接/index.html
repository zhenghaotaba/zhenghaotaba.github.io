数据库设置远程连接

Jul 29, 2019
1 min read

在学习或者工作中我们会遇到需要别人访问自己数据库的情况，大致有两种方法：
一、将库做备份，在需要访问的电脑上恢复库
二、直接远程连接到自己的库
如果在需要数据同步的情景下，方法一就不适用了。
以常见的关系型数据库为例

Oracle

将被连接的数据库TNSLSNR服务添加到防火墙允许程序中即可。

Mysql

登入mysql后，更改”mysql” 数据库里的 “user” 表里的 “host”项，从”localhost”改称”%“。
示例代码：

use mysql;

select 'host' from user where user='root'

update user set host='%' where user='root'

flush privileges;

select 'host'   from user where user='root';
